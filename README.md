# Optimisation-strat-gique-du-march-freelance-international-Upwork-Data-Analysis-
Analyse du marché freelance (64k offres Upwork) enrichie par salaires et coût de la vie. Création du FAR pour comparer le pouvoir d’achat réel. Dashboard interactif (carte, Sankey, Sunburst) pour aider freelances à se positionner et recruteurs à optimiser coûts et sourcing à l’international.
